# Self Directed Study Repository

This repo serves as a grouped storage space for resources and projects used to further develop my knowledge of C/C++ and Embedded Systems, the objective of this repo is to enable me to pivot into the Embedded industry.

## Books

[C Programming: A Modern Approach](https://archive.org/details/c-programming-a-modern-approach-2nd-ed-c-89-c-99-king-by/mode/2up):
* This has been used extensively to familiarise myself with the C language, reading and following the given problems within the book to program and solidify understanding of the content.

## Online Resources

Below are some online resources used during the process of self-teaching lower level programming.

[Daniel Hirsch (Youtube)](https://youtube.com/@HirschDaniel):
* An insightful and idea provoking channel centered around the creation of challenging projects using bare minimum technologies/ frameworks (E.g. Vim, standard C libraries and documentation alone).

[Hoff_World (Youtube)](https://youtube.com/@hoff._world):
* A young but experienced embedded systems engineer, highly capable at breaking down complex concepts into palatable information. Mostly covers higher level aspects of the embedded world with a few hands on projects mixed in.

[Gene Schroeder Tech](https://youtube.com/@geneschroedertech7501):
* Published to youtube in 2024, this contains a 26 lesson long course on embedded programming, being titled 'Bare Metal Embedded Programming: Theory and Practice Using STM32'. My intention is to complete this course after having finished reading Dr. K.N. King's book and becoming proficient enough to not be tripped up the fundamentals of Mr Schroeder's course.
